# BEMA Website

Production Astro site for BEMA Incorporated, a Florida nonprofit healthcare access coordination organization.

## Stack

- Astro static site generation
- Tailwind CSS
- TypeScript
- Markdown content collections
- Vercel deployment target

## Commands

```sh
npm install
npm run dev
npm run build
npm run preview
```

## Project Notes

- Active routes: `/`, `/about`, `/services`, `/providers`, `/blog`, `/get-started`, `/contact`, `/employers`
- Legal routes: `/privacy-policy`, `/terms`
- Content collections: `blog`, `team`, `siteInfo`, with reserved inactive collections retained for future use
- Primary CTA wording for the partner clinic route is `Find a Partner Clinic`
- Contact form action is a placeholder until a deployment form handler is selected

## Compliance Notes

BEMA coordinates access to care through independent partner clinics. It does not deliver care directly. Public copy should preserve that distinction and keep human-authored content in bracketed placeholders until approved.
